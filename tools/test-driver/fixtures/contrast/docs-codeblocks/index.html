Getting started

This guide walks through installing the toolkit and writing your first program. Every code sample below uses syntax highlighting, which means each block contains a spread of small colored tokens sitting on a lightly tinted grey background. Those colored tokens are exactly the kind of low-contrast-risk text a theme engine tends to ruin.

Installation

Install the package with your favorite manager, then import it at the top of your entry file:

// install first: bun add toolkit
import { greet } from "toolkit";

function main() {
  const message = greet("world");
  console.log(message);
}

Configuration

The toolkit reads a small config object. Keys you do not set fall back to sensible defaults, so a minimal setup is only a few lines long:

/* every field below is optional */
export const config = {
  retries: 3,
  timeout: 5000,
  label: "production",
  onError: reportError,
};

With that in place you can run the program and watch the output. If the colored tokens above are still legible after the theme engine has done its work, the harness will report a passing contrast score for this page.
