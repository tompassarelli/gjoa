A logo drawn entirely in SVG text

The wordmark above is not an image file — it is live SVG, with the brand name set in fill="currentColor" so it tracks the surrounding text color, and a small tagline pinned to a hardcoded fill="#333". Both of these are real text nodes that a contrast harness can read and score, even though they live inside a vector graphic rather than ordinary HTML flow.

This matters for dark mode because SVG text follows different inheritance rules than HTML text. A run painted with currentColor will move along with whatever the engine does to the inherited color, while a run pinned to an explicit hex value will only change if the inverter reaches into the SVG presentation attributes. Getting both cases right is the whole point of this fixture.

Surrounding everything is plain HTML body copy, so the page mixes ordinary paragraph text with the two SVG text styles. After inversion all three should remain comfortably readable against the new background.
