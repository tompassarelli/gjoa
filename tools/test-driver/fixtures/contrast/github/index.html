octocat / hello-world

This repository declares a color-scheme of light dark and ships a real dark theme behind a prefers-color-scheme media query. A browser that respects native dark support should render this page using the site's own dark palette and leave the inversion engine out of it entirely.

The README explains the project in a few short paragraphs. Because the site author already did the work of choosing legible foreground and background pairs for dark mode, the contrast harness should find that every text box here passes the accessibility threshold without any help from the luminance inverter.

If the engine instead inverts a site that already provides its own dark theme, the result is a double negative: dark text on a dark background, or washed-out grey on grey. Detecting and honoring the native theme is exactly what the force-native tier is meant to guarantee.
